Woot! A new Epic for Mac (Arm-based chips) :-).

      Please follow these steps to update Epic:

      1. Download the new Epic: click here to download

      2. Run the Downloaded File


      3. Drag the Epic icon into the Applications folder


      4. Select 'Replace' to replace the old Epic with the new Epic
      (you won't lose saved bookmarks or passwords)
_________________________________________________________________________

      Important Note regarding Keychain access. With any update of Epic, you may see a popup requesting
      Keychain access. It is for Epic-only data (no other Keychain data). Default Chromium behavior is
      to store passwords there. It will come up each time you open Epic (sorry, we know it's annoying,
      it's Mac/Chromium default behavior and not our choice) unless you give it "always allow" access.

      Remember, never install Epic from any website other than epicbrowser.com
